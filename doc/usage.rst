Guide d'utilisation
===================

Cas d'utilisation 1 : Consulter un bien
---------------------------------------
1. L'utilisateur accède à la page d'accueil.
2. Clique sur le bouton "Lettings".
3. Sélectionne un bien pour voir l'adresse détaillée.

Cas d'utilisation 2 : Rechercher un profil
------------------------------------------
1. L'utilisateur clique sur "Profiles".
2. Sélectionne un utilisateur pour voir sa ville favorite.
