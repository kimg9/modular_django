Interfaces de programmation (API)
=================================

L'application expose des points de terminaison pour la consultation des données.

Endpoints principaux
--------------------

* ``GET /profiles/`` : Liste tous les profils.
* ``GET /profiles/<username>/`` : Détails d'un profil spécifique.
* ``GET /lettings/`` : Liste tous les biens à louer.
* ``GET /lettings/<id>/`` : Détails d'un bien.
