Description du Projet
=====================

Le projet consiste en une plateforme web permettant aux utilisateurs de consulter 
des biens immobiliers à louer. Chaque bien est associé à un profil (propriétaire ou agent).

Objectifs principaux :
----------------------
* Centraliser les annonces de location.
* Faciliter la recherche de biens par profil.
* Offrir une interface d'administration robuste via Django.
