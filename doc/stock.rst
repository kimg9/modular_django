Technologies et Langages
========================

Le projet s'appuie sur une pile technologique éprouvée, optimisée pour la scalabilité et le déploiement continu.

* **Langage :** Python 3.11 (supporté via la matrice de build GitHub Actions)
* **Framework Web :** Django
* **Base de données :** SQLite3
* **Conteneurisation :** Docker (Image hébergée sur Docker Hub)
* **Orchestration :** AWS Elastic Container Service (ECS)
* **CI/CD :** GitHub Actions
* **Monitoring :** Sentry
